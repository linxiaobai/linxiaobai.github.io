环境

操作系统：linux(精确点是centos)
Mysql安装目录：/var/lib/mysql
Mysql备份目录：/bada_sql_backup （开始没有此目录）

步骤

1.创建保存备份文件的路径/bada_sql_backup

mkdir / bada_sql_backup

2.创建/usr/sbin/bakmysql文件

vim /usr/sbin/bakmysql
输入（注意下面日期的引号比较特殊，千万不要输错）

或者写成

3.修改文件属性，使其可执行

chmod +x /usr/sbin/bakmysql

4.修改/etc/crontab
vim /etc/crontab

输入


（表示每天3点钟执行备份）
5.重新启动crond

/etc/rc.d/init.d/crond restart

6.Finish
